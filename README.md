# Atividades de Lógica JavaScript PROA

Projeto desenvolvido durante o curso **PROA — Mais que empregos, futuros**, com foco em prática de **lógica de programação**, **JavaScript**, **HTML** e **CSS**.

A aplicação reúne uma grande coleção de exercícios interativos organizados por temas e módulos, permitindo praticar desde operações básicas até estruturas de repetição, seleção, listas e manipulação de arquivos.

---

## 📌 Sobre o projeto

Este projeto foi criado como uma plataforma de अभ्यास/treino de lógica de programação no navegador, utilizando apenas tecnologias front-end.

A proposta é facilitar o aprendizado por meio de exercícios práticos, onde cada atividade pode ser executada diretamente na interface, com uso de `prompt`, `alert` e funções JavaScript.

Além disso, o projeto possui visual estilizado, navegação entre páginas e alternância entre **tema escuro e tema claro**.

---

## ✨ Funcionalidades

- Interface web simples e interativa
- Mais de **120 exercícios de JavaScript**
- Organização por categorias e módulos
- Exercícios de:
  - variáveis
  - operadores aritméticos
  - estruturas condicionais
  - operadores lógicos
  - laços de repetição
  - listas
  - manipulação de arquivos
- Sistema de alternância entre **tema claro/escuro**
- Persistência do tema escolhido via `localStorage`
- Navegação entre páginas:
  - `index.html`
  - `exercicios.html`
  - `contato.html`
- Página de contato com links para redes sociais
- Estilo visual personalizado com identidade forte e aparência retrô

🚀 Como executar o projeto

Como este é um projeto front-end estático, não precisa instalar dependências.

Opção 1: Abrir diretamente
Baixe ou clone o repositório
Abra o arquivo index.html no navegador
Opção 2: Usar extensão de servidor local

Se preferir, você pode usar uma extensão como Live Server no VS Code para rodar o projeto localmente.
